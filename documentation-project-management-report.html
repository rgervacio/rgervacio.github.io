Project Description

Project Shore Line: Documentation Project includes a content plan, budget, and schedule (with a Gantt chart) for a hypothetical online banking service’s documentation project.

This document was created to fulfill the requirements for Simon Fraser University's TCOM320: Documentation Project Management course. In TCOM320, students were required to create a documentation project plan for a fictitious credit union’s online banking software application. To do this, students pretended to be project managers leading a documentation team responsible for creating customer support materials. The course covered topics such as documentation content planning, team and resource management, and agile project management principles.
